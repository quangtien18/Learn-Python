Đầu tiên bạn phải tải Python về,bạn vào trang : python.org

Bạn vào phần Downloads và tải về python bản mới nhất , hiện tại là bản 3.6.5

Nếu bạn dùng hệ điều hành window thì tải về như trên sau đó giải nén ra và chạy file . Bạn mở phần tìm kiếm trong cửa sổ window gõ python chạy file và bắt đầu sử dụng python .

Nếu bạn dùng hệ điều hành ubuntu , bạn mở terminal lên và gõ lệnh :

apt-get install python

Enter và gõ password máy tính của bạn vào , khí đó máy sẽ tự tải và cài đặt python cho bạn . Sau khi cài đặt xong bạn gõ python nó sẽ hiện ra >>> , tức là bạn đã cài thành công và bắt đầu sử dụng python .

Mình đang dùng hệ điều hành ubuntu , ubuntu sẽ cho bạn cảm giác giống như một lập trình viên , tức là đa số các thao tác của bạn sẽ là gõ lệnh nhưng cũng yêu cầu bạn cũng phải học lệnh . Điều cần biết thêm là nếu bạn có chơi game trên máy tính thì hãy dùng hệ điều hành window vì ubuntu không hỗ trợ nhiều trong việc chơi game . Mình đã phải bỏ chơi game rất nhiều từ khi học lập trình , mặc dù game là món ăn tinh thần của mình :*( .

Tại đây mình cũng giới thiệu thêm một ứng dụng rất hữu ích và mình nghĩ là nên có , đó là sublime text 3

Bạn click vào đây để tải sublime text về .

Tại sao nên dùng Sublime text , bởi vì :

  • Là phần mềm trả phí, nhưng cho phép dùng thử vô thời hạn.
  • Chạy quá nhẹ khởi động nhanh, tốn ít tài nguyên so với một full IDE mà vẫn thường được dùng.
  • Hỗ trợ trên các hệ điều hành phổ biến như Windows, Mac OS và các bản phân phối của Linux như Ubuntu, Linux Mint,...
  • Có một giao diện người dùng trực quan, dễ dàng tùy biến, rất bắt mắt.
  • Hỗ trợ nhận diện và tô màu từ khoá (syntax highlight) cho hơn 20 ngôn ngữ khác nhau chẳng hạn như HTML, CSS, Javascript, C++, C#, SQL, XML ….
  • Bạn có thể thêm những tính năng mới thông qua việc cài đặt Plugin hay Package (hoàn toàn miễn phí).

Khi đã cài đặt xong mọi thứ và bắt đầu sử dụng , ban đầu sẽ hơi khó khăn nhưng rồi sẽ quen thôi . Chúc bạn thành công .
