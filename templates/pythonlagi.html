Python là gì ?

Python là ngôn ngữ lập trình thông dịch do Guido van Rossum tạo ra năm 1990.Khi bạn học tiếng Anh,tiếng Hàn,tiếng Trung,...bạn gọi đó là học ngôn ngữ,thì python cũng vậy nó cũng là một loại ngôn ngữ và gọi là ngôn ngữ lập trình.Nó được lập ra để làm mối liên kết giữa ta và máy tính,giữa ta và máy tính và những người khác sao cho các đối tượng hiểu nhau,hiểu ta đang làm cái gì,ta viết ra cái gì đó cho máy tính hiểu và người khác hiểu.

Python chạy trên mọi hệ điều hành và là ngôn ngữ bậc cao .

Ngoài python ra còn rất nhiều ngôn ngữ lập trình khác như Ruby , C# , PHP , JavaScrip , ... rất nhiều.Và mỗi ngôn ngữ đều vô cùng rộng , vì vậy bạn nên chọn một thứ ngôn ngữ và học hoặc chỉ cần học một ngôn ngữ thôi là đã đủ .

Một số ngôn ngữ lập trình phổ biến hiện nay :

  • JavaScript
  • Java
  • C#
  • Python
  • PHP

Tính tới năm 2016 python chiếm 26,7% trong các ngôn ngữ và hiện tại tới tương lai chắc chắc đây sẽ là ngôn ngữ phát triển mạnh.

Tại sao nên học python :

  • Cú pháp dễ đọc dễ hiểu
  • Thư viện phong phú
  • Cộng đồng sử dụng lớn

  2. Lập trình game
